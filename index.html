Analyzing the Impact of Hybrid Sorting Algorithms on Energy Usage in Python Applications

This research project investigates how different hybrid sorting algorithms impact energy consumption in Python applications. The study focuses on understanding how algorithm design, input size, input characteristics, and data types influence energy efficiency.

View Source Explore Dataset

Research Overview

With the increasing demand for energy-efficient software, this research evaluates the performance of hybrid sorting algorithms under controlled experimental conditions. Energy usage and execution time were measured using Intel Power Gadget. The results provide valuable insights into selecting optimal algorithms for energy-aware computing.

Research Objectives

  • Evaluate and compare the energy consumption of hybrid sorting algorithms
  • Analyze the impact of input size and input order on energy usage
  • Examine differences between integer and floating-point data types
  • Study the relationship between execution time and energy consumption

Dataset

The dataset used in this research contains detailed energy measurements collected during algorithm execution. It includes information such as algorithm type, input size, input distribution, execution time, energy consumption, and data type (integer and floating-point).
